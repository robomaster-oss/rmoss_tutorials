# RMOSS 代码规范

#### 代码风格规范

采用[ROS2代码规范](https://docs.ros.org/en/foxy/Contributing/Code-Style-Language-Versions.html).

C++:
* 文件命名,变量命名,函数命名采用`snake_case`(下划线)方式.
* 类,结构体命名采用`CamelCase`(大驼峰)方式.

> 更多内容查看ROS2代码规范文档
